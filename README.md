# Claude Code Usage Analytics Platform

This project implements an end-to-end analytics pipeline for Claude Code telemetry data.

## Features

- Telemetry ingestion from JSONL logs
- Data processing and transformation
- Analytics aggregation
- Interactive Streamlit dashboard
- Developer usage insights

## Architecture

Pipeline:

Telemetry Logs → Processing → Aggregation → Dashboard

Technologies used:

- Python
- Pandas
- Streamlit
- Plotly
